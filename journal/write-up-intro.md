# Making A Checkers Game with Unit Tests using C++ and SFML

## Introduction
Recently for fun and as a probably needed refresher from my coursework, I started
reading the book [*A Programmer's Introduction to Mathematics*](https://pimbook.org/)
by Jeremy Kun. In the intro to the first chapter he briefly mentioned that in his
undergrad one of his projects a C++ checkers game that generates a list of legal
jump-moves from a given board state. I thought that sounded like a really fun
project and a great way to exercise my admittedly rusty C++ chops. His description
was a good jumping off point, but for this to be a truly fun project I think that
there needs to be some added features.

## Project Specifications:
### My implementation:
- must have graphics and be controlled using mouse point and clicks
- should be playable, not just a board state input
- should be networked, allowing for me to play a game with my girlfriend.
- should still show the previously mentioned legal set of jump moves for every
  board state

That should be a pretty good list of features that will keep me busy for quite a
while!
